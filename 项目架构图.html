三、双层检索流程 (Dual-Layer Retrieval)

graph TD
    Q["🔍 用户查询
'违法解除劳动合同赔偿标准'"] --> QT["📝 Query Understanding
提取争议焦点"]
    QT --> QE["🔢 查询向量化
task_type=RETRIEVAL_QUERY"]

    QE --> L1["📚 第一层：文档召回
Qdrant legal_documents
top_k=5"]
    L1 --> L2["📖 第二层：段落召回
Qdrant legal_paragraphs
按 doc_id 过滤"]
    L2 --> RR["🔄 Reranker 重排序
BGE-Reranker-V2-M3
交叉编码器精排"]
    RR --> EX["💡 匹配解释生成
DeepSeek V3 API
说明段落相关性"]
    EX --> VAL["✅ 溯源校验
citation 完整性检查
snippet 提取"]
    VAL --> RES["📊 返回结果
文档 + 段落证据 + 引用"]
