# 法律文书相似检索系统 - 启动说明

## 启动方式

### 1. 确保 Docker 运行

先启动 Docker Desktop，然后确认 PostgreSQL 和 Qdrant 容器在运行：

```bash
docker start legal-search-postgres legal-search-qdrant
```

### 2. 启动后端

```bash
cd "E:\fl app\backend"
"D:\Anaconda\envs\legal-search\python.exe" -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 3. 启动前端

```bash
cd "E:\fl app\frontend"
"D:\Anaconda\envs\legal-search\python.exe" -m http.server 3000
```

## 相关链接

| 名称 | 地址 |
|---|---|
| 前端界面 | http://localhost:3000 |
| 后端 API | http://localhost:8000 |
| API 文档 (Swagger) | http://localhost:8000/docs |
| 健康检查 | http://localhost:8000/api/health |
| Qdrant 控制台 | http://localhost:6333/dashboard |

## 快速检查命令

```bash
# 查看所有服务状态
curl http://localhost:8000/api/health

# 手动触发初始化（首次启动后运行一次）
curl -X POST http://localhost:8000/api/bootstrap
```

## 一键启动（可选）

也可以直接用项目自带的脚本启动后端（会自动用 conda 环境）：

```bash
cd "E:\fl app\backend"
start.bat
```

前端同理需要单独开一个终端启动。
